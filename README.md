## About

This landing page is designed for a laundry and dry cleaning business, providing customers with essential information about services, pricing, and contact details. The website features a clean, professional design optimized for both desktop and mobile devices.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Service Showcase**: Detailed presentation of laundry and dry cleaning services
- **About Section**: Information about the business, experience, and commitment to quality
- **Contact Information**: Easy-to-find phone number and email with clickable links
- **Professional Imagery**: High-quality images from Unsplash showcasing laundry services
- **Semantic HTML5**: Proper document structure with accessibility in mind
- **Fast Loading**: Optimized static site with minimal dependencies
- **SEO Friendly**: Proper meta tags and semantic markup for search engine optimization

## Technology Stack

- **HTML5**: Semantic markup for structure and content
- **CSS3**: Modern styling with flexbox/grid layouts and responsive design
- **Vanilla JavaScript**: Interactive features without framework dependencies
- **No Build Process**: Pure static files for easy deployment and maintenance
